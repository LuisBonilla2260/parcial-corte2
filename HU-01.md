o# Historia de Usuario - HU-01

## **Título:** Seleccionar Fecha y Hora para Reservas

### **Descripción:**
Como usuario del sistema, quiero poder seleccionar una fecha y hora específicas para realizar una reserva, de manera que pueda planificar mi visita al establecimiento.

---

## **Criterios de Aceptación:**

1. **Visualización del Componente:**
   - El sistema debe mostrar un campo para seleccionar la fecha y hora.
   - El campo debe estar etiquetado como "Fecha y Hora".

2. **Interacción:**
   - El usuario debe poder abrir un selector de fecha y hora al hacer clic en el campo.
   - El selector debe permitir elegir tanto la fecha como la hora en un formato amigable.

3. **Validación:**
   - El sistema debe registrar correctamente la fecha y hora seleccionadas.
   - Si no se selecciona una fecha y hora, el valor debe permanecer vacío.

4. **Compatibilidad:**
   - El componente debe ser funcional en dispositivos móviles y de escritorio.

---

## **Definición de Hecho:**
- El componente de selección de fecha y hora está implementado y probado.
- La funcionalidad es accesible desde la página de gestión de reservas.
- El valor seleccionado se almacena correctamente en el estado del componente.

---

## **Diseño del Componente:**

### **Archivo:** `SeleccionFechaHora.tsx`
```tsx
import React, { useState } from 'react';
import { IonItem, IonLabel, IonDatetime } from '@ionic/react';

const SeleccionFechaHora: React.FC = () => {
  const [fechaHora, setFechaHora] = useState<string | undefined>();

  return (
    <IonItem>
      <IonLabel position="stacked">Fecha y Hora</IonLabel>
      <IonDatetime
        value={fechaHora}
        presentation="date-time"
        onIonChange={(e) => {
          const valor = e.detail.value;
          if (typeof valor === 'string') {
            setFechaHora(valor);
          } else if (Array.isArray(valor)) {
            setFechaHora(valor[0]); // toma el primer valor si es arreglo
          }
        }}
      />
    </IonItem>
  );
};

export default SeleccionFechaHora;
```
