# Historia de Usuario - HU-02

## **Título:** Customer Data Component Aggregation

### **Descripción:**
Como usuario del sistema, quiero poder ingresar los datos del cliente (nombre y número de contacto) para realizar una reserva, de manera que el establecimiento pueda identificar al cliente y contactarlo si es necesario.

---

## **Criterios de Aceptación:**

1. **Visualización del Componente:**
   - El sistema debe mostrar campos para ingresar el nombre y el número de contacto del cliente.
   - Cada campo debe estar claramente etiquetado.

2. **Interacción:**
   - El usuario debe poder escribir el nombre y el número de contacto en los campos correspondientes.
   - El campo de contacto debe aceptar únicamente números.

3. **Validación:**
   - El sistema debe validar que el campo de contacto contenga solo números.
   - Si el campo de contacto contiene caracteres no válidos, no se debe permitir su ingreso.

4. **Compatibilidad:**
   - El componente debe ser funcional en dispositivos móviles y de escritorio.

---

## **Definición de Hecho:**
- El componente de ingreso de datos del cliente está implementado y probado.
- La funcionalidad es accesible desde la página de gestión de reservas.
- Los datos ingresados se almacenan correctamente en el estado del componente.

---

## **Diseño del Componente:**

### **Archivo:** `IngresoDatosCliente.tsx`
```tsx
import React, { useState } from 'react';
import { IonItem, IonLabel, IonInput } from '@ionic/react';

const IngresoDatosCliente: React.FC = () => {
  const [contacto, setContacto] = useState('');

  const handleInputChange = (event: any) => {
    const value = event.target.value;
    if (/^\d*$/.test(value)) {
      setContacto(value);
    }
  };

  return (
    <>
      <IonItem>
        <IonLabel position="stacked">Nombre</IonLabel>
        <IonInput placeholder="Ingrese su nombre" />
      </IonItem>
      <IonItem>
        <IonLabel position="stacked">Contacto</IonLabel>
        <IonInput
          placeholder="Ingrese su número de contacto"
          value={contacto}
          onIonInput={handleInputChange}
        />
      </IonItem>
    </>
  );
};

export default IngresoDatosCliente;
```
